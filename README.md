# crystal-secp256k1-zkp

crystal lang binding for secp256k1-zkp.

## Installation

1. Add the dependency to your `shard.yml`:

   ```yaml
   dependencies:
     crystal-secp256k1-zkp:
       github: syalon/crystal-secp256k1-zkp
   ```

2. Run `shards install`

## Usage

```crystal
require "crystal-secp256k1-zkp"
```
